# bank-network-simulation
Toy model of banks financial dependencies on each other using an exposure matrix and an SIR style visualization

This project contains code generated by ChatGPT.

In this brief project we create a toy simulation of a banking network, and aim to visualize it using an SIR-system. The initial idea was to use an SIR model, but exposures are not random, so we use an n*n matrix instead, to store exposures.
